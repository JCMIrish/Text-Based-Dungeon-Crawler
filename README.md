## Project Title: TBDC - A simple Text Based Adventure

Description: A text based adventure where you stumble into a dungeon slay a few foes and confront the King of Rot. Do you have the Strength to face such a foe?

Developed by: JCMIrish                                                                                         
License: MIT                                                                                                    

## Features
- Multiple rooms to explore
- Simple Equipment System (Equip and Unequip gear)
- Simple Shop with equipment, selling of trinkets or gear, and healing potions
- Gold from enemies slain
- A boss fight at the end
- Score system which tracks items, gold, and remaining HP.
